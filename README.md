# Undeadremains
 
This is a Minecraft Mod adding a whole bunch of new stuff for making the Minecraft experience more fun.
This mod is undead themed to add a mystical aspect...

This mod was built ontop a modding template by **Kaupenjoe**. 

You should be able to build the mod yourself by using ./gradlew on POSIX systems and ./grable.bat on Windows

To find out more about Diverging Lense Studios who built and maintaines the Mod, <a href="https://www.youtube.com/channel/UC1XPopjhTSCfnNQerHS0AkA">Click here!</a>
